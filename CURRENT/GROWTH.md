# GROWTH — Estrategia de Crecimiento

> Mantenido por Perplexity + Rodhan. Claude Code NO modifica este archivo salvo instrucción explícita.
> Última actualización: 2026-06-30 | 📝 Provisional

## Estado comercial actual
- Fase: Desarrollo — 0 clientes activos | ERP multi-tenant con IA nativa (DeepSeek)
- Modelo: SaaS B2B | Stack: Rust + PostgreSQL multi-tenant + DeepSeek API

## Cliente objetivo
- PyMEs peruanas (10-100 empleados): manufactura, retail, servicios
- Necesidad: ERP accesible con IA integrada (vs SAP/Odoo complejos)
- Precio: S/ 199-499/mes por empresa (schema dedicado + IA queries)

## Top 3 canales de adquisición
1. **LinkedIn B2B**: casos de uso "PyME migró de Excel a Valdhran en 1 semana"
2. **Alianzas contadores/consultores**: comisión por referral de clientes
3. **Google Ads**: "ERP para PyME Perú", "alternativa SAP barata"

## Diferenciador clave
- IA nativa: consultas lenguaje natural sobre inventario/ventas (DeepSeek API)
- Multi-tenant desde día 1: cada cliente schema PostgreSQL dedicado
- Rust performance: queries rápidas vs ERPs tradicionales (Node.js/PHP)

## Hitos de crecimiento
- [ ] Primer cliente piloto activo (1 empresa, 3 meses gratis)
- [ ] Demo pública funcional (inventario + ventas + IA query)
- [ ] Alianza con 1 contador/consultor empresarial
- [ ] 5 clientes pagando → validar modelo pricing
- [ ] Módulo contabilidad Perú (SUNAT integración)
